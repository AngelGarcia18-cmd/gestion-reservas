# EventEase
## Descripción
**EventEase** es un sistema integral de gestión de eventos y reservas desarrollado en Java con NetBeans que permite gestionar reservas de eventos, clientes y espacios de manera eficiente.
El proyecto incluye una interfaz intuitiva y funcionalidades para crear, modificar y eliminar resevas, así como consultar información de clientes y eventos. Además de facilitar la organización de eventos tanto para pequeñas como grandes organizaciones.

---

## Funcionalidades principales

- **Gestión de Clientes:** agregar, editar, eliminar y consultar información de clientes.
- **Gestión de Eventos:** crear eventos, asignar fechas y hora.
- **Gestión de Reservas:** registrar reservas de clientes a eventos, actualizar y eliminar reservas.
- **Historial y reportes:** consultar histórico de reservas y eventos, generar reportes básicos.
- **Interfaz gráfica (GUI):** fácil de usar, desarrollada con Java.
- **Seguridad básica:** control de acceso y validaciones de datos.

---

## Tecnología utilizadas

- **Lenguaje:** Java
- **IDE:** NetBeans
- **Control de versiones:** Git y GitHub
- **Base de datos:** MySQL
- **Otros:** JDBC para conexión con base de datos

## Instalación

1. Clonar el repositorio:
   ```bash
   git clone https://github.com/AngelGarcia18-cmd/eventease.git
2. Abrir el proyecto en NetBeans:
   • Archivo → Abrir Proyecto → Seleccionar la carpeta del proyecto.
3. Configurar la base de datos.
4. Ejecutar el proyecto desde NetBeans.

## Uso

• Ejecutar la aplicación desde NetBeans.
• Usar la interfaz para:
  • Registrar clientes y participantes.
  • Crear y administrar eventos.
  • Gestionar reservas y consultar historial.
• Navegar por las distintas secciones para gestionar clientes, eventos y reservas.
• Guardar los cambios para mantener la información actualizada.
